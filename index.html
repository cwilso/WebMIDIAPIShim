Web MIDI API Polyfill

This is a polyfill using the Jazz NPAPI MIDI plugin to implement the Web MIDI API on Mac and Windows.

Download this project as a .zip file Download this project as a tar.gz file

To use the Web MIDI API Polyfill

Just include

<script src='http://cwilso.github.com/WebMIDIAPIShim/WebMIDIAPI.js'></script>

in your HTML source.

Test it out!

Before you press the button: install the Jazz-Soft.net's Jazz-Plugin to enable MIDI support on Windows and OSX.

On MacOS, you will also need a real synthesizer or something like MidiKeys.
